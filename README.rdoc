== README

Chariot helps friends share rides. 
Our app automatically finds your geolocation and matches you with friends heading in the same direction. 

Technologies used:

* Ruby 4.2.0

* Rails API

* Geocoder Gem 1.2.7 for location information

* Google Places API

* Authentication written from scratch on server and client side with Bcrypt

* AngularJS

* Sass

* Heroku link https://chariots.herokuapp.com

* Trello link https://trello.com/b/wZDlwPRc/chariot
